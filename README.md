Lab-2 Битүүмжлэл
Энэхүү лабораторийн ажлаар битүүмжлэл болон мэдээлэл нууцлалтыг практик жишээн дээр хэрэгжүүлж сурсан. Анхны кодонд Renderer класс нь тодорхой дүрс болох Rectangle-д шууд хамааралтай, мөн дүрсүүдийн гишүүн өгөгдлүүд public байсан.
Үүнийг засахын тулд Shape нэртэй интерфейсийг тодорхойлж, бүх дүрсүүд уг интерфейсийг хэрэгжүүлдэг болгосон. Мөн дүрсүүдийн дотоод өгөгдлийг private болгож, зөвхөн шаардлагатай getArea() методоор дамжуулан ашигладаг болсон. Ингэснээр дүрсүүдийн хэрэгжүүлэлтийн мэдээлэл нууцлалт сайжирсан. Мөн Renderer класс нь тодорхой дүрсээс хамаарах бус, зөвхөн Shape интерфейсээс хамаарах байдлаар өөрчлөгдсөнөөр кодын уян хатан байдал нэмэгдсэн. Үүний үр дүнд шинэ дүрс нэмэхэд Renderer классыг өөрчлөх шаардлагагүй болсон.
